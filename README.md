# Portfolio Web

A Minimalist Full-Stack Portfolio.

## Tech Stack
- **Languages**: JavaScript
- **Frontend**: React.js, Vite
- **Backend**: Supabase (PostgreSQL)
- **Libraries/Tools**: Clerk (Authentication), Framer Motion, React Three Fiber, Three.js

## Active Features
- Minimalist Card-based Layout with Glassmorphism
- 3D Interactive Elements & Background Animations (AnimatedBlob, FallingEmoji, MeshGradientBackground)
- Realtime Admin Panel for Project Management
- Media Gallery & Full-screen Lightbox
- AI ChatBot Assistant (Claude 3 / Anthropic API)
