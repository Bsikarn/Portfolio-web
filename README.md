# Portfolio Web

A Minimalist Full-Stack Portfolio.

## Tech Stack
- React.js (Vite)
- Supabase (PostgreSQL with Realtime)
- Clerk (Authentication)
- Framer Motion (Animations)
- React Three Fiber / Three.js (3D Graphics)

## Active Features
- Minimalist Card-based Layout with Glassmorphism
- 3D Interactive Elements & Background Animations (AnimatedBlob, FallingEmoji)
- Realtime Admin Panel for Project Management
- Media Gallery & Full-screen Lightbox
